# Grafana Dashboard

## NYC Taxi Analytics - Uber/Lyft FHVHV July 2025

Place the exported dashboard JSON here as `nyc-taxi-dashboard.json`.

### How to export:
1. Go to [laxmanbarredata.grafana.net](https://laxmanbarredata.grafana.net)
2. Open "NYC Taxi Analytics - Uber/Lyft FHVHV July 2025"
3. Click Share icon → Export tab → Save to file
4. Rename the file to `nyc-taxi-dashboard.json` and place it in this folder

### Dashboard Panels:
1. Peak Hours Bar Chart (hour of day vs trip count)
2. Daily Trip Volume Line Chart (July 2025 trend)
3. Trip Duration Breakdown (Pie/Bar - Under 10min, 10-20min, 20-30min, Over 30min)
4. Shared vs Solo Rides (97% solo dominance)
5. Driver Pay vs Platform Cut Time Series (~$6 platform cut per trip)
